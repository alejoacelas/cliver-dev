## Section 2: Narrative

### What this check is and how it works

USPS RDI is a binary flag (Residential / Commercial) derived from the USPS Address Management System. It is the canonical postal classification of every US deliverable address. Direct integration requires CASS-certified software and hash-table licensing from USPS, which is impractical for most synthesis providers. In practice, the check is consumed through CASS-certified vendors like Smarty (formerly SmartyStreets), which return the RDI field as part of their standard US address verification response. Critically, the Smarty API call that returns RDI is the same call that returns the CMRA flag used by m03-usps-rdi-cmra, so for providers already running M03 checks, M04's RDI check costs nothing additional at the margin. When `metadata.rdi` returns "Residential" on an order claimed against an institutional affiliation, the system flags the order for human review.

### What it catches

RDI catches the same bypass methods as the county assessor parcel use-code check: every residential-address variant across the mapped attacker stories. This includes the community-bio-lab-network's garage-lab-at-home, the dormant-domain story's $0 residential colocation, the foreign-institution's US-based residential home office (Method 4), and the residential edge cases in cro-framing and gradual-legitimacy-accumulation. The hit/miss profile is identical to m04-county-assessor across every attacker story in the mapping. The stage 5 hardening assessment explicitly notes this redundancy: RDI adds no bypass methods caught that county assessor misses.

### What it misses

Every commercial-address bypass method passes RDI cleanly — virtual offices, coworking spaces, maker spaces, executive suites, biotech incubator benches, and freight forwarders all classify as "Commercial." International addresses produce zero signal. The uncovered bypass list is identical to the county assessor's. The structural limitation is that RDI is a single binary bit with no nuance: it cannot tell you what kind of residential address it is (home vs. converted loft vs. apartment), and it cannot tell you what kind of commercial address it is (real lab vs. virtual office).

### What it costs

RDI via Smarty is among the cheapest checks in the pipeline: estimated $0.001–$0.005 per lookup, and effectively $0 marginal for providers already calling Smarty for m03's CMRA check (same API response). The Smarty free tier allows entry-level testing. Setup cost is approximately one day for API integration; Smarty account provisioning is self-service. Compared to the county assessor check (Regrid bulk license at $80K/year), RDI is orders of magnitude cheaper for the same binary residential/commercial signal.

### Operational realism

The manual review handoff is institution-class-aware: a residential flag on an order from a major university is treated as a hard flag (request institutional shipping address), while the same flag on a small startup or community-bio LLC triggers a carve-out review with requests for supporting documentation (lab photos, biosafety committee letter, fume hood permits). The implementation correctly cross-references RDI against county assessor data where available, using the richer land-use codes to add context to the binary flag. The audit trail is straightforward: the Smarty JSON response, the binary RDI value, and the reviewer's disposition are logged in the compliance file. One operational concern: since RDI's false-positive population is a strict subset of county assessor's, running both checks on the same address doubles the review paperwork without surfacing additional information — the review playbook should merge these flags into a single review step.

### Open questions

Smarty's per-call pricing ($0.001–$0.005) is a best guess based on historical public information and G2 reviews; the 04C claim check flagged this as UPGRADE-SUGGESTED, recommending that a v2 pull the actual number from Smarty's pricing calculator. The Smarty free tier was described as "~250 lookups/month" but the 04C check noted this number is from memory — the exact tier should be verified. Smarty's ToS for screening use is unresolved (searched for but not found); no known restriction exists, but this has not been confirmed. RDI accuracy for mixed-use buildings at the unit level is estimated at 85–95% but has no external citation.
